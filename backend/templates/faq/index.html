База знаний (FAQ)

Быстрые ответы на главные вопросы по платформе

Платформа использует точный мэтчинг на основе введенных навыков (тегов) и грейда. Вы не увидите информационный шум — только целевые предложения. Компании из вашего Черного списка автоматически исключаются из выдачи на уровне БД.

Откройте страницу конкретной вакансии от компании, которую хотите заблокировать. В правой панели действий (под контактами) нажмите на кнопку с иконкой перечеркнутого круга "В черный список".

В данный момент API находится в стадии закрытого бета-тестирования . Если вы хотите получить ранний доступ для автоматической выгрузки вакансий из своей ATS-системы, напишите нам в саппорт.
